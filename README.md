Empty Geddy Project with Socket-io
==================================



Install Node-js:

http://nodejs.org/download/


Install geddy with npm:
<pre>
	$ npm install -g geddy
</pre>
Open the empty_project dir and type
<pre>
$ npm install
</pre>

Geddy tutorial:
http://geddyjs.org/tutorial
